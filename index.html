What It Does

Compares a Return from Manufacturing (RFM) slip against your Purchase Order file to verify quantities — showing exactly how many units were ordered vs how many are being physically returned, per part number. Parts with no matching PO entry are flagged separately. Duplicate PO rows for the same part are automatically summed before comparison.

Requirements

One RFM file containing the "Return Mat Slip" sheet. One PO file containing the "PO_Status" sheet. Both uploads can be the same file — the tool reads each sheet independently. All formats supported.

1 RFM file 1 PO file .xlsx .xls .xlsb
RFM File Format

Sheet name: "Return Mat Slip". Rows 1–9 are meta/header rows and are automatically skipped. Data begins at row 13. Expected columns:

Col A = Item Req No  ·  Col B = Make  ·  Col C = Type/Order No (Part Number)  ·  Col D = Description  ·  Col E = Scope  ·  Col F = Section  ·  Col G = Return Qty  ·  Col H = Unit For Qty

Sheet: Return Mat Slip Data from row 13 Col C → Part No Col G → Return Qty
PO File Format

Sheet name: "PO_Status". Rows 1–4 are header/meta rows. Data begins at row 5. Expected columns:

Col D = Type/Order No (Part Number)  ·  Col I = PO Qty  ·  Col J = PO No  ·  Col K = Supplier  ·  Col L = Pending Qty. If the same part number appears on multiple rows, all PO Qty values are summed automatically.

Sheet: PO_Status Data from row 5 Col D → Part No Col I → PO Qty Duplicates summed
Project Details (Optional)

Step 1 includes a Project Details panel with six fields — PM Name, HW Team Lead, Client Name, Order Value, Projected Material Cost, and Actual Material Cost. These are optional and only populate the Summary sheet inside the exported Excel.

PM Name HW TL Client Name Order Value Projected Cost Actual Cost
What You Get

A 2–3 sheet Excel export (.xlsx):

"RFM Report" sheet — full comparison table with all RFM columns plus matched PO data: Item Req No, Make, Part No, Description, Scope, Section, Customer, OR No, Project Qty, PO Qty, PO No, Supplier, Used Qty, Return Qty, % Returned, Unit Price, Total Amt Returned. Each row is colour-coded by match status.

"Summary" sheet — project KPIs with auto-calculated cost percentages. "Unmatched Parts" sheet — only included when RFM contains parts with no matching PO entry, flagging what was returned but never ordered.
